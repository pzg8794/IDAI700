# IDAI700 Archive and Research Foundation

This repository archives the full working directory for `IDAI700: Ethics of AI` and preserves
the materials that shaped later fairness-oriented research projects.

## What this repo contains

- `IDAI700-Course_Info/`: syllabus, assignment sheets, due-date exports, and course logistics
- `IDAI700-Unit_*/`: unit readings, slides, lecture materials, reflections, and related files
- `Research Paper Portfolio/`: the independent research portfolio, source PDFs, and the final
  AI ethics paper
- `Reflections/`: reflection papers, notes, and supporting course artifacts
- `DSCI601_Ethics_Fairness_Report/`: a later writing workspace that adapts the IDAI700 ethics
  paper into a DSCI601 ethics/fairness report focused on fairness-aware sequential decision-making

## Main authored paper in this archive

The strongest foundation document in this repo is:

- `Research Paper Portfolio/IDAI700_Research_Paper.pdf`

That paper develops the argument that AI systems affecting marginalized communities should be
designed with enforceable accountability, not only aspirational ethics language. In later work,
that idea becomes one of the motivations for centering fairness in sequential decision-making.

## Why this repo is public

This repo serves two purposes:

1. Preserve the original IDAI700 course archive and research trail.
2. Provide a documented source base for later fairness-oriented writing and project framing.

## Notes

- This repo includes a mix of course-distributed files and authored work.
