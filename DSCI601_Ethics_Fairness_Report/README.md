# DSCI601 Ethics and Fairness Report Workspace

This folder adapts the IDAI700 AI Ethics paper into a DSCI601 one-page ethics/fairness report.

## Files

- `requirements.md`: exact assignment requirements and project-specific interpretation
- `TODO.md`: working checklist for turning the ethics foundation into the final DSCI601 report
- `source_lineage.md`: what was carried forward from the IDAI700 paper and what was intentionally left behind
- `ethics_fairness_report.tex`: one-page Overleaf-ready draft

## Foundation source

The main conceptual source for this report is:

- `../Research Paper Portfolio/IDAI700_Research_Paper.pdf`

The DSCI601 report does not reuse that paper verbatim. It extracts the parts that motivate why
fairness became central to the current project design.
